Shallow Foundation Design
--------------------------

q_ult = c Nc + q Nq + 0.5 G B Ng

Soil Type
..........

  - Cohesive soils (phi=0): c*(2+pi) + q*1 + 0
  - Non-Cohesive soils: The embedment effect is particularlyimportant.

Correction Factors
...................

  - Footing shape
  - Ground water table
  - Embedment depth
  - Sloping

Local or Punching Shear
.......................

  - c* = 0.67 c
  - phi* = 1/tan(0.67*tan(phi))
